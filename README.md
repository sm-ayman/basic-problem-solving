# JavaScript Problem Solving

A collection of beginner-friendly JavaScript problem-solving exercises covering fundamental programming concepts such as loops, arrays, strings, conditionals, and algorithms. Each solution is implemented in JavaScript and focuses on efficient problem-solving techniques.

## Problems Included

### 1. Prime Number Check

Determine whether a given number is prime.

**Input:** `29`
**Output:** `Prime Number`

### 2. Find Largest Element in Array

Find the largest element in an array efficiently.

**Input:** `[12, 45, 7, 89, 23]`
**Output:** `89`

### 3. Reverse a String

Reverse a string without using built-in reverse functions.

**Input:** `"algorithm"`
**Output:** `"mhtirogla"`

### 4. Palindrome Check

Check whether a string is a palindrome.

**Input:** `"madam"`
**Output:** `Palindrome`

### 5. Factorial Calculation

Calculate the factorial of a given number.

**Input:** `5`
**Output:** `120`

### 6. Find Duplicate Elements

Identify duplicate elements in an array.

**Input:** `[1, 3, 5, 3, 7, 1]`
**Output:** `[1, 3]`

### 7. Swap Two Numbers Without a Third Variable

Swap two numbers without using an additional variable.

**Input:** `a = 10, b = 20`
**Output:** `a = 20, b = 10`

### 8. Count Vowels in a String

Count the total number of vowels in a string.

**Input:** `"Programming"`
**Output:** `3`

### 9. Find Second Largest Element

Find the second largest value in an array.

**Input:** `[10, 50, 30, 80, 70]`
**Output:** `70`

### 10. Remove Duplicates from Array

Remove duplicate values and return unique elements.

**Input:** `[1, 2, 2, 3, 4, 4, 5]`
**Output:** `[1, 2, 3, 4, 5]`

## How to Run

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:

   ```bash
   cd javascript-problem-solving
   ```

3. Run any solution:

   ```bash
   node filename.js
   ```

This repository is designed to strengthen problem-solving skills, improve algorithmic thinking, and prepare for coding interviews using JavaScript.
